# Code of Conduct

## Our Standard

This project aims to be welcoming, respectful, and constructive.

Examples of behavior that contribute to a positive environment include:

- showing empathy and patience
- giving actionable, good-faith feedback
- assuming good intent while still being clear and honest
- focusing criticism on ideas, code, and behavior rather than people

Examples of unacceptable behavior include:

- harassment or personal attacks
- discriminatory or exclusionary language
- deliberate intimidation or trolling
- publishing private information without permission

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and other public interactions around the project.

## Enforcement

If you experience or witness unacceptable behavior, please open a private channel with the maintainer if possible rather than escalating publicly first.

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this Code of Conduct.
